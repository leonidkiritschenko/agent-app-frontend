# Requirement Document

## Introduction
- Web front-end app with API to the backend
    - Text and speech input
    - Text and media output

## Requirements

### Requirement 1

**Story:** 
A user expects basic function of chat app. Type in text into prompt field. Press send button to get a response from LLM. Prompt and response showing up as chat log.

**Acceptance Criteria:**
1. User can type in a prompt into prompt-field.
2. User can press send-button to send prompt to LLM.
3. User sees the prompt in chat-log.
4. User sees response from LLM in the chat-log.
